### Highlighted repositories

* **Developing-a-CT-to-MRI-Translation-with-BBDM**
  Brownian Bridge Diffusion Model (BBDM) pipeline for CT→MRI translation with style-key conditioning, multi-slice inference, and ISTA-based refinement for stable volumetric reconstruction. Includes evaluation using SSIM, PSNR, and NRMSE. 

* **Habitat-Robotics-NavSim**
  Robotics navigation and exploration experiments in simulated environments with heuristic search, object reasoning, and environment understanding.

* **Diffusion Models & Optimal Transport**
  Neural Optimal Transport methods for accelerating diffusion model sampling through latent-space truncation, including comparative experiments with truncation strategies, variational autoencoders (VAEs), diffusion models, and Wasserstein-based probabilistic metrics.

* **VL-MemKnG**
  Video QA for long-horizon egocentric navigation videos using segment-level memory retrieval and caption-derived spatio-temporal knowledge graphs. Combines GraphRAG-style reasoning with multimodal retrieval to support evidence-grounded question answering across temporally scattered events.
